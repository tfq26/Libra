# Product Overview

Libra is a technical support chatbot application that provides AI-powered assistance for technological issues. The system consists of:

- **Frontend**: User interface for chat interactions (referenced but not present in current workspace)
- **Backend**: Azure Functions-based API that handles chat conversations and integrates with Azure OpenAI

## Core Features

- Interactive chat interface with AI-powered technical support
- Conversation history management and persistence
- Image upload support for visual troubleshooting
- User authentication via Clerk
- Conversation summarization and titling

## Target Users

End users seeking technical support assistance through a conversational AI interface that can provide step-by-step troubleshooting guidance.
